# Source Preview

## 1. 목적
`Source Preview`는 운영자가 수집 소스의 원문이 제대로 파싱되는지 확인하는 도구입니다.

## 2. 호출 방식
- 엔드포인트: `POST /api/v1/ingestion/crawl/preview`
- 웹 화면: 운영자 도구 영역의 `파라미터 입력` 버튼 클릭 -> 슬라이드 입력 패널 -> 실행

## 3. 입력
- `source_id`
- `max_chars`

## 4. 출력
- 파싱 결과 JSON
- 본문 추출 결과
- 일부 메타 정보

## 5. 목적별 사용 예시
- HTML/PDF/DOCX/HWP 문서가 제대로 파싱되는지 확인
- 특정 소스가 메뉴 페이지를 수집하는지, 실제 문서를 수집하는지 점검
- 배치 실행 전 파싱 로직 품질 점검

## 6. 서비스 위치
이 기능은 사용자 제품이 아니라 운영자 도구입니다.
