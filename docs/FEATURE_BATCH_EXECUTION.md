# Batch Execution

## 1. 목적
운영자 화면에서 주요 배치 수집 작업을 수동 실행하기 위한 기능입니다.

## 2. 현재 제공 배치 예시
- 시장 체제 스냅샷 배치
- 네이버 섹션 헤드라인 배치
- 정책브리핑/한국은행 문서 배치
- 글로벌 거시 브리핑 배치
- 공개 리서치 리포트 배치

## 3. 화면 역할
- 배치 버튼 클릭으로 즉시 실행
- 성공/실패 여부와 요약 결과 확인
- 운영자가 데이터 신선도를 강제로 갱신할 때 사용

## 4. 활용 시점
- 서버 기동 직후 주요 데이터 선적재
- 외부 소스 지연이나 구조 변경 점검
- `Market Regime` 초기 응답 속도 개선을 위한 사전 적재
