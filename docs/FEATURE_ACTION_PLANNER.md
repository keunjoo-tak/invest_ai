# Action Planner

## 1. 목적
`Action Planner`는 `Stock Decision` 결과를 기반으로 실제 행동 계획을 세우기 위한 서비스입니다.

## 2. 입력
- 종목명 또는 티커
- 투자 기간
- 위험 성향
- 보유 여부
- 평균 매입가
- 투자 목적

## 3. 주요 출력
- 상단 행동 결론
- 행동 판단의 전제 조건
- 가격 구간 설계
- 실행 로드맵
- 보유 여부별 분리 계획
- 투자 기간별 계획
- 위험 성향별 커스터마이즈
- 시나리오별 실행 전략
- 행동 근거 설명
- 실행 전 체크리스트 및 무효화 조건
- 원문 및 출처 문서

## 4. 활용 데이터
- `Stock Decision`의 시그널과 근거 데이터
- 이벤트 타임라인
- 공시 점수화 결과
- 리서치 문서와 섹터 비교 결과

## 5. 리포트 특징
- 입력 조건에 따라 행동 문구가 달라짐
- 가격 구간, 위험 구간, 체크포인트를 분리해 제시
- 관련 뉴스, 공시, 리서치 문서의 출처 열기 지원
- 단계형 로딩 리포트 제공

## 6. 사용 시점
- 매수 계획을 세울 때
- 보유 종목 대응 전략을 정할 때
- 장전 체크리스트가 필요할 때
