# Action Planner 리포트

## 1. 목적
`Action Planner`는 종목 판단 결과를 실제 행동 계획으로 바꾸는 서비스입니다.

핵심 질문은 다음과 같습니다.
- 지금 신규 진입을 고려할 구간인가?
- 보유 중이면 어떤 조건에서 계속 들고 가야 하는가?
- 무효화 구간과 목표 구간은 어디인가?

## 2. 호출 방식
- 엔드포인트: `POST /api/v1/action-planner/analyze`
- 웹 화면: `조건 입력 후 실행` 버튼 클릭 -> 슬라이드 입력 패널 -> 실행

## 3. 입력
- `ticker_or_name`
- `investment_horizon`
- `risk_profile`
- `has_position`

## 4. 출력 리포트 구조
### 첫 영역
- 최종 행동 권고
  - 예: `삼성전자에 대한 현재 권고는 관찰 유지입니다.`

### 두 번째 영역
- 추천 행동
- 종목 판단 결과
- 판단 점수
- 품질 점수

### 세 번째 영역
- 매수 관심 구간
- 무효화 구간
- 목표 구간
- 실행 전 확인할 조건
- 보유자 계획
- 미보유자 계획
- 시나리오별 실행 전략
- 데이터 출처

## 5. 활용 데이터
- 직접적으로는 `Stock Decision` 결과를 사용
- 간접적으로는 시세, 뉴스, 공시, 재무제표, 거시, 정책 문서가 모두 반영됨

## 6. 처리 흐름
1. `Stock Decision` 결과 조회
2. 투자 기간과 위험 성향을 반영해 행동 문장 생성
3. 진입/무효화/목표 구간 계산
4. 상방/중립/하방 시나리오 구성
5. 리포트로 출력

## 7. 사용자 신뢰 포인트
- 단순 매수/매도 표현 대신 실행 구간과 무효화 조건 제시
- 보유자와 미보유자의 계획을 분리해 설명
- 종목 판단 결과를 그대로 재사용하지 않고 행동 관점으로 다시 구조화
