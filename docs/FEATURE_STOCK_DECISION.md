# Stock Decision

## 1. 목적
`Stock Decision`은 특정 종목의 현재 판단을 리포트 형식으로 제공하는 서비스입니다. 종목 자체의 가격 흐름, 이벤트, 공시, 재무, 섹터, 거시 환경을 함께 반영합니다.

## 2. 입력
- 종목명 또는 티커

## 3. 주요 출력
- 상단 핵심 결론
- 판단 점수 구조 분해
- 상승/하락 요인
- 기간별 해석
- 시장-섹터-종목 계층 분석
- 가격/수급/기술 해석
- 이벤트·뉴스·공시 해석
- 재무/실적/가치 판단
- 거시/정책 영향
- 체크포인트 및 반증 조건
- 데이터 신뢰 및 설명 가능성

## 4. 활용 데이터
- KIS 시세 및 가격 파생 변수
- 네이버 검색 뉴스, NewsAPI
- OpenDART 공시와 재무제표
- 섹터 모멘텀 및 대장주 커플링
- 장전 미국 증시 영향
- 정책/거시 관련 데이터 중 종목 관련성이 높은 항목
- 공개 리서치 리포트

## 5. 리포트 특징
- 이벤트 타임라인 카드에서 출처 열기 지원
- 수시공시 점수화 카드에서 원문 공시 링크 제공
- 리서치 컨센서스 카드에서 출처 열기 지원
- 섹터 peer 비교 표 제공
- 세부 섹션은 접기/펼치기 가능

## 6. 사용 시점
- 신규 진입 판단 전
- 보유 종목 재점검 시
- 특정 공시나 뉴스가 나온 직후
