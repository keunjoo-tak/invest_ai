# 데이터 카탈로그

## 1. 목적
이 문서는 현재 프로젝트에서 수집하거나 관리하는 데이터가 어디에서 오고, 어떤 서비스에서 어떤 용도로 사용되는지 빠르게 파악하기 위한 문서입니다.

## 2. 종목 및 시세 데이터
### 2.1 종목 마스터
- 원천 소스: KRX/DART 종목 식별 정보
- 설명: 종목명, 티커, 시장, 영문명, 별칭, 섹터 분류에 필요한 기본 정보
- 활용 용도: 종목 식별, 섹터 유니버스 생성, 대장주 매칭
- 전처리: 종목명 정규화, 별칭 확장, 시장 구분 보정
- 사용 서비스: 전체 서비스

### 2.2 시세 데이터
- 원천 소스: 한국투자증권 KIS API
- 설명: 일봉 시세, 거래량, 거래대금
- 활용 용도: 추세, 수익률, ATR, 갭 수익률, 변동성, 거래대금 강도 계산
- 전처리: 결측 제거, 수익률 계산, 이동평균/ATR 파생
- 사용 서비스: `Stock Decision`, `Action Planner`, `Watchlist Alerts`

## 3. 뉴스 데이터
### 3.1 종목 관련 뉴스
- 원천 소스: 네이버 검색 API, NewsAPI
- 설명: 특정 종목명 또는 영문명을 기준으로 검색한 기사
- 활용 용도: 이벤트 타임라인, 종목 관련 뉴스 근거, 감성 및 키워드 밀도 계산
- 전처리: 종목명 직접 언급 여부 확인, 주가 영향 문맥 점수화, 무관 기사 제거
- 사용 서비스: `Stock Decision`, `Action Planner`, `Watchlist Alerts`

### 3.2 네이버 섹션 헤드라인
- 원천 소스: 네이버 뉴스 섹션 페이지 직접 크롤링
- 설명: 정치, 경제, IT/과학, 세계 섹션의 헤드라인 기사
- 활용 용도: 최근 5일 시장 헤드라인 영향 설명
- 전처리: 링크 추출, 본문 파싱, LLM 요약, 섹션별 통합 영향 정리
- 사용 서비스: `Market Regime`

## 4. 공시 및 재무 데이터
### 4.1 공시 데이터
- 원천 소스: OpenDART, KIND 연계 공시 정보
- 설명: 수시공시, 재무 관련 공시, 사업보고서 메타
- 활용 용도: 이벤트 타임라인, 공시 점수화, 중대 이벤트 탐지
- 전처리: 공시 유형 분류, LLM/규칙 기반 호재·악재 점수화, 관련성 필터링
- 사용 서비스: `Stock Decision`, `Action Planner`, `Watchlist Alerts`

### 4.2 재무제표 데이터
- 원천 소스: OpenDART 재무제표 API
- 설명: 매출, 영업이익, 순이익, 자산, 부채, 현금흐름 등
- 활용 용도: 수익성, 안정성, 성장성, 유동성 판단
- 전처리: 계정 정규화, 비율 계산, YoY 성장률 계산
- 사용 서비스: `Stock Decision`, `Action Planner`

## 5. 국내 거시 및 정책 데이터
### 5.1 국내 거시 지표
- 원천 소스: BOK, KOSIS
- 설명: 환율, 금리, CPI, 수출, 생산 등 국내 거시 지표
- 활용 용도: 시장 체제 해석, 종목 관련 거시 근거 선별
- 전처리: 표준 거시 row 변환, surprise index 계산, 관련성 필터링
- 사용 서비스: `Market Regime`, `Stock Decision`

### 5.2 정책브리핑 및 한국은행 문서
- 원천 소스: 정책브리핑, 한국은행 공개 자료
- 설명: 정책 발표문, 브리핑, 조사자료, 간행물
- 활용 용도: 시장 영향 이벤트, 정책 방향 해석
- 전처리: 상세 본문/첨부문서 추출, 원문 저장, LLM 요약, 시장 영향 선별
- 사용 서비스: `Market Regime`

## 6. 글로벌 거시 데이터
- 원천 소스: OECD, FRED, BLS, BEA, Fiscal Data, IMF, World Bank, Eurostat
- 설명: 금리, 인플레이션, 고용, 성장률, 재정, 국제 비교 지표
- 활용 용도: 글로벌 거시 환경 판단, 위험선호/위험회피 해석
- 전처리: 표준 거시 row 변환, surprise index 계산, 발표 시점 메타 관리
- 사용 서비스: `Market Regime`, 일부 `Stock Decision`

## 7. 글로벌 이슈 및 공식 이벤트
- 원천 소스: 공식 캘린더, 공식 릴리즈, 글로벌 이슈 스트림
- 설명: 주요 발표 일정과 광범위한 정책/시장 이슈
- 활용 용도: 이벤트 변동성 모드, 리스크 경고, 시장 체제 보조 근거
- 전처리: 이벤트 분류, 시점 정렬, 유사 이벤트 패턴 매칭
- 사용 서비스: `Market Regime`, `Stock Decision`

## 8. 리서치 리포트
- 원천 소스: 국내 증권사, 국내 은행·연구소, 글로벌 자산운용사 공개 리포트
- 설명: 데일리 시황, 투자 전략, 산업/기업 분석 리포트
- 활용 용도: 시장 체제 해석, 종목 컨센서스 근거
- 전처리: 문서 다운로드, 본문 추출, LLM 요약, 종목/시장 단위 태깅
- 사용 서비스: `Market Regime`, `Stock Decision`, `Action Planner`

## 9. 파생 특징 변수
- 설명: 원천 데이터를 바로 쓰지 않고 예측과 판단에 적합한 수치로 변환한 변수 집합
- 예시: 수익률, ATR, 거래대금 Z-score, 뉴스 주목도, 공시 순효과, 거시 surprise, 섹터 커플링, 미국 야간 전이 계수
- 사용 서비스: `Stock Decision`, `Action Planner`, `Watchlist Alerts`

## 10. 저장 위치
- 데이터베이스: 핵심 정규화 데이터와 배치 적재 결과 저장
- 다운로드 폴더: 원문 PDF, HTML, DOCX 등 파일 보관
- 로그 폴더: 배치 실행 로그, 외부 공개 접속 로그 보관
