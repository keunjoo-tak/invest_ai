# Market Regime

## 1. 목적
`Market Regime`는 오늘 시장이 어떤 상태인지 요약하고, 현재 유리한 대응 방식과 주의해야 할 리스크를 설명하는 서비스입니다.

## 2. 입력
- 별도 입력 없음
- 호출 시점 기준 최신 배치 데이터와 실시간 수집 결과를 활용

## 3. 주요 출력
- 상단 핵심 결론
- 시장 체제 분해 점수
- 핵심 원인 Top 5
- 섹터 판단
- 투자자 행동 연결
- 반증 조건
- 최근 5일 섹션별 헤드라인 영향
- 리서치 체제 해석

## 4. 활용 데이터
- 국내/글로벌 거시 지표
- 정책브리핑, 한국은행 자료
- 글로벌 이슈 스트림
- 네이버 뉴스 섹션 헤드라인
- 공개 리서치 리포트
- 시장 대표 종목 및 섹터 스냅샷

## 5. 리포트 특징
- 리포트 내용은 섹션 카드로 나뉘며 접기/펼치기가 가능
- 헤드라인 카드에서는 상세 보기와 출처 열기 기능 제공
- 리서치 카드에서는 원문 출처 링크 제공

## 6. 사용 시점
- 장 시작 전 오늘의 시장 대응 방향을 정할 때
- 장중에 섹터 강약과 매크로 환경을 다시 점검할 때
- `Stock Decision` 실행 전 상위 시장 맥락을 먼저 확인할 때
