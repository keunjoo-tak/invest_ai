# Watchlist Alerts

## 1. 목적
`Watchlist Alerts`는 관심 종목의 즉시 점검 필요 여부를 확인하고, 저장형 감시 목록을 관리하는 서비스입니다.

## 2. 입력
- 종목명 또는 티커
- 알림 채널

## 3. 주요 출력
- 즉시 점검 여부
- 주요 트리거
- 리스크 플래그
- 촉매 요인
- 알림 미리보기
- 근거 원문 및 출처

## 4. 활용 데이터
- 종목 분석 핵심 시그널
- 최근 뉴스와 공시
- 이벤트 점수화 결과
- 저장된 구독 정보

## 5. 기능
- 단건 점검
- 구독 생성
- 구독 목록 조회
- 구독 삭제
- 관련 문서 출처 열기

## 6. 사용 시점
- 장중에 빠르게 종목 상태를 확인할 때
- 반복 점검이 필요한 종목을 저장할 때
